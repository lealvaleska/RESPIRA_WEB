# Respira

Projeto desenvolvido para a disciplina de Front-end do curso de Sistemas para Internet.

O Respira é um site de acolhimento digital com foco em saúde mental, pensado para oferecer uma experiência mais leve, acessível e acolhedora ao usuário.

O projeto utiliza conceitos de UX emocional, psicologia das cores e regras de negócio para adaptar os conteúdos conforme o estado emocional informado pelo usuário.

## Funcionalidades

- Triagem emocional
- Recomendações personalizadas
- Exercícios de respiração
- Diário emocional
- Modo acessível
- Informações e canais de apoio

## Regras de negócio

O sistema altera conteúdos e prioridades de acordo com as respostas do usuário.

Exemplos:
- usuários em crise recebem contatos emergenciais em destaque;
- conteúdos são recomendados conforme o estado emocional;
- registros emocionais frequentes podem gerar sugestões de apoio psicológico.
