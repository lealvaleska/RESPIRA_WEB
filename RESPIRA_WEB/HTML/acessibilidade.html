Modo Acolhedor

O Modo Acolhedor foi criado para proporcionar uma experiência mais calma e confortável. Quando ativado, o site se adapta para reduzir estímulos visuais e melhorar a legibilidade.

O que muda quando ativado:

📝

Fonte maior

Tamanho do texto aumentado para melhor leitura

🎨

Cores mais suaves

Paleta de cores ainda mais relaxante

✨

Menos animações

Redução de movimentos e transições

📏

Mais espaçamento

Maior distância entre elementos

👁️

Melhor contraste

Textos mais legiveis e claros

💚

Visual acolhedor

Ambiente mais calmo e confortável

Nosso Compromisso com Acessibilidade

Recursos implementados para tornar o Respira acessível a todos

⌨️

Navegação por Teclado

Todo o site pode ser navegado usando apenas o teclado. Use Tab para navegar entre elementos e Enter para ativar.

📖

Leitores de Tela

Estrutura semântica e atributos ARIA para compatibilidade com tecnologias assistivas.

📱

Design Responsivo

Layout adaptável a diferentes tamanhos de tela, desde celulares até desktops.

🎨

Contraste Adequado

Cores escolhidas para garantir legibilidade seguindo as diretrizes WCAG.

🔍

Textos Escaláveis

Fontes que podem ser aumentadas sem perda de funcionalidade ou quebra de layout.

🎯

Áreas de Clique Amplas

Botões e links com tamanho adequado para facilitar a interação.

Dicas de Conforto Digital

💡

Ajuste o brilho da tela

Reduza o brilho em ambientes escuros para diminuir a fadiga ocular.

⏰

Faça pausas regulares

A cada 20 minutos, olhe para algo a 20 metros por 20 segundos (regra 20-20-20).

🌙

Use o modo noturno

Ative o filtro de luz azul do seu dispositivo a noite.

📐

Posicione a tela corretamente

Mantenha a tela a cerca de um braco de distancia e na altura dos olhos.
